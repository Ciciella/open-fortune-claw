## Codebase Patterns
- This project uses AdonisJS framework with path aliases (#services, #controllers, etc.)
- The bot services (trend_bot.ts, grid_bot.ts, arbitrage_bot.ts) use a singleton CONFIG pattern
- TypeScript type errors for module resolution are pre-existing and not blocking for runtime
